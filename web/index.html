Registered Devices
1. Register Devices
Edit devices.py on the Pi:
DEVICES = [ { "name": "my-machine", "ip": "192.168.1.10", "description": "My desktop", "tags": ["linux"], "admin": True, # sees all devices }, { "name": "server-01", "ip": "192.168.1.20", "description": "Home server", "tags": ["linux"], # no admin — sees own chat only }, ]
2. Start Hub (Pi5)
uvicorn server.app:app \ --host 0.0.0.0 --port 8000
3. Local Agent
Drop local_agent/local_agent.py on each machine.
BRICK_PROJECT_ROOT=/path/to/brick \ python local_agent.py
4. Systemd Service
[Service] Environment=BRICK_PROJECT_ROOT=/opt/brick ExecStart=/usr/bin/python3 \ /opt/brick/local_agent/local_agent.py Restart=always
