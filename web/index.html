1. Define Your Devices
Edit devices.py on the server:
DEVICES = [ { "name": "my-machine", "ip": "192.168.1.10", "description": "My desktop", "tags": ["linux"], "admin": True, # gets sidebar + device panel }, { "name": "server-01", "ip": "192.168.1.20", "description": "Home server", "tags": ["linux"], # no admin flag — just their own chat }, ]
2. Start Hub
uvicorn server.app:app \ --host 0.0.0.0 --port 8000
3. Run Local Agent
Copy local_agent/local_agent.py to each machine.
BRICK_PROJECT_ROOT=/path/to/brick \ python local_agent.py
4. Run as Service
[Unit] Description=Brick Local Agent After=network.target [Service] Environment=BRICK_PROJECT_ROOT=/opt/brick ExecStart=/usr/bin/python3 \ /opt/brick/local_agent/local_agent.py Restart=always [Install] WantedBy=multi-user.target
