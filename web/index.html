Setup
Configured Devices
Enter admin key in the header.
1. Define Your Devices
Edit devices.py on the server:
DEVICES = [ { "name": "raspberrypi", "ip": "192.168.1.10", "description": "Living room Pi", "tags": ["pi", "home"], }, ... ]
2. Start Hub
BRICK_ADMIN_KEY=yourpassword \ uvicorn server.app:app \ --host 0.0.0.0 --port 8000
3. Run Local Agent
Copy local_agent/local_agent.py to each machine. No keys needed — the hub identifies it by IP.
BRICK_PROJECT_ROOT=/path/to/brick \ python local_agent.py
4. Run as Service
[Unit] Description=Brick Local Agent After=network.target [Service] Environment=BRICK_PROJECT_ROOT=/opt/brick ExecStart=/usr/bin/python3 \ /opt/brick/local_agent/local_agent.py Restart=always [Install] WantedBy=multi-user.target
